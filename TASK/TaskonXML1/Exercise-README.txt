We will be doing XSL Transformation in this exercise : 
----------------
Requirement : 
----------------

1. Transform Input xml to WorkOrder.xml

2. There are fields which are required, which are mentioned in the WOrkOrder xml [Those fields are mandatorily present]

3. You will be writing and xsl file for transforming.

4. The mappings for the fields present in the Workorder xml is provided in the workOrder xml itself as a note besides the fields.

5. Based on the notes you need to figure out which fields are to be mapped with the xpath from the input xml.


To test your XSL Transformation has been made correct or not, you can check online tools, google it.
